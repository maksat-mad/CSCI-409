# Application for an online store.

This project was created using React / Redux

### Role division
1. User
2. Admin
3. Super Admin

### Features

|    Roles    | Change personal information | Buy/sell products | Leave reviews | Block/unblock users | Promote/demote Admins |
|:-----------:|:---------------------------:|:-----------------:|:-------------:|:-------------------:|:---------------------:|
|    User     |              +              |         +         |       +       |          -          |           -           |
|    Admin    |              +              |         -         |       -       |          +          |           -           |
| Super Admin |              +              |         -         |       -       |          +          |           +           |
